Fundamentos da Linguagem Java

Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90. Alguns conceitos fundamentais:

  • Sintaxe básica e estrutura de programas Java
  • Tipos de dados primitivos e wrappers
  • Operadores e estruturas de controle
  • Arrays e coleções básicas
// Exemplo de código Java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Programação Orientada a Objetos

Os quatro pilares da POO em Java:

  1. Abstração: Classes e objetos
  2. Encapsulamento: Modificadores de acesso
  3. Herança: Extends e implements
  4. Polimorfismo: Sobrecarga e sobrescrita

Tratamento de Exceções

Como lidar com erros em Java:

  • try-catch-finally
  • Checked vs Unchecked exceptions
  • Criando exceções personalizadas

HTML5 Semântico

HTML5 introduziu elementos semânticos que melhoram a estrutura e acessibilidade:

  • <header>, <nav>, <main>, <footer>
  • <article>, <section>, <aside>
  • <figure> e <figcaption> para imagens
  • Formulários avançados com novos tipos de input
<!DOCTYPE html>
<html>
<head>
  <title>Página Semântica</title>
</head>
<body>
  <header>
    <h1>Meu Site</h1>
  </header>
  <nav>...</nav>
  <main>...</main>
</body>
</html>

CSS3 Moderno

Recursos avançados do CSS3 para layouts modernos:

  1. Flexbox: Layout unidimensional
  2. Grid: Layout bidimensional
  3. Animações: @keyframes e transitions
  4. Variáveis CSS: Para valores reutilizáveis

Design Responsivo

Técnicas para criar sites que funcionam em qualquer dispositivo:

  • Media queries
  • Unidades relativas (em, rem, vh, vw)
  • Imagens responsivas
  • Mobile-first design
